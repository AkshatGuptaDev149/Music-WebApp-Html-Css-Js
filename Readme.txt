How to add Songs?
Ans. First go script.js and just look for a array of objects named 'songs'.
2nd. Just add the song details like in this example
"{songName:'Bleach OST Treachery',filePath:'songs/Aizen theme.mp3',coverPath:'coverImages/cover1.png'}"

type your songname,filepath and coverPath in the given fields

That's it you are good to go
